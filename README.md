# 💰 Expense Tracker
A simple and efficient Expense Tracker application built using Java to help users manage their daily expenses. The application allows users to add, update, delete, and view expenses while generating monthly and category-wise reports. It provides an easy-to-use menu-driven interface for tracking personal spending and analyzing expenses.

## ✨ Features
* ➕ Add new expenses
* ✏️ Update existing expenses
* 🗑️ Delete expenses
* 📋 View all expenses
* 🔍 Search expenses
* 📅 Generate monthly expense reports
* 🏷️ Generate category-wise reports
* 📈 Calculate total expenses
* 💸 Display highest expense
* 💰 Display lowest expense
* 🖥️ User-friendly console interface

## 🛠️ Technologies Used
* Java
* Object-Oriented Programming (OOP)
* ArrayList
* HashMap
* VS Code / IntelliJ IDEA
* Git & GitHub

## 🚀 How to Run
1. Clone this repository.
2. Open the project in VS Code or IntelliJ IDEA.
3. Compile the Java files.
4. Run the `Main.java` file.
5. Follow the on-screen menu to manage expenses.
